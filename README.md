# How to

How To — Projeto Flutter
App móvel desenvolvido em Flutter como parte da disciplina Programação de Dispositivos Móveis na faculdade. O projeto foi estruturado para explorar o desenvolvimento de interfaces e lógica de negócio em Dart/Flutter, suportando múltiplas plataformas como Android, iOS e Web.

Se trata de um aplicativo móvel contendo uma gama variada de tutoriais dos mais variados temas, bem como instruçoes em texto, imagem e video explicando/ensinando o usuário a realizar tarefas.

Possui sistema de autenticação utilizando firestore, banco de dados não relacional firebase, integraçao com Google ML para detecção de rostos, no intuito de fornecer maior acessibilidade para pessoas com deficiência visual, etc

O objetivo principal é demonstrar habilidades práticas com:

Flutter (Dart) para construção de UI reativa multiplataforma

Organização de pastas e modularização do código

Testes básicos e configuração de dependências

Estruturas de dados, navegação de telas e uso de assets
